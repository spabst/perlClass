Perl - moose classes generation support

A small web-App that wants to make developper's life easier

This web application has been developed as a small IT project at the University of Lausanne. The main purpose of the present work is to accelerate the creation of an object oriented project under Perl. We all know that Perl 5 is not a native Object Oriented language. A large number of modules have been created to enable a similar programming experience. One of this packages is Moose, a very diffused and popular package that should work, as close as possible, like the upcoming version 6 of Perl, that we have been waiting for a long time and should finally be natively Object Oriented.
